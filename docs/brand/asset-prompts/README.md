## How the swap works

1. Generate the asset in ChatGPT using the matching prompt (audit memo has the
   full self-contained prompt text per slot; the audit's hex/hard-NO block is
   inlined in every one).
2. Save it to the **swap path** (create `public/brand/plaino-system/scenes/` if
   absent).
3. In `components/ui/ap/PlainoScene.tsx`, change the slot's right-hand value in
   the `SRC` map from `${PLACEHOLDERS}/<slug>.svg` to the real path string.
4. For OG cards, point `heritageUrl` in the OG route at the new asset.

## Brand lock (every prompt repeats this)

Palette ONLY: Paper `#F7F4ED` · Ink `#1A1A1F` · Clay `#B65D3A` (one charge) ·
Moss `#3F5C3F` (verified/passed only) · Mute `#8C8478` (distance/haze).
Hard NO: no blue · no pure black/white · no gradients · no AI-sparkle/glow ·
no glassmorphism · no isometric · no drone/aerial · no suited people · no text
in the image · Plaino is calm/faithful, never aggressive, never glossy mecha.
