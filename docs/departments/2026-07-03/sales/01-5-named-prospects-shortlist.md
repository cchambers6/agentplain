# 01 — The five named prospects (Georgia RE, first-touch cohort)

**Date researched:** 2026-07-03, from public sources only (firm websites, LinkedIn company pages, directory listings — each row cites its sources). No internal or private data; nothing here is scraped from anywhere a prospect couldn't see themselves.
**Format:** fills the worksheet in `docs/outreach/2026-07-03-design-partner-kit/00-cohort-shortlist.md`. ICP + disqualifiers per `docs/sales/deep-dive-2026-07-02/01-named-icp-per-vertical.md`: independent brokerage, ~5–40 agents, broker-owner active in operations, Georgia-first.

**Honesty rules for this sheet:**
- **Referral path:** no referrer is named because no referrer has agreed to be named. Every row says *cold pending warm-map pass* — Conner works his FlatSBO-adjacent map (closing attorneys, lenders, past colleagues) against these five names before the block; any hit upgrades that row to variant (a) and jumps the queue.
- **Verify before send:** agent counts and roles below come from public pages as of 07-03. The Monday block's step 2 re-confirms each hook is still true and each broker is still the broker (GREC license lookup is free). A stale fact = fix the row, not skip the send.
- **LinkedIn:** only URLs that actually surfaced in research are listed. Where the personal profile wasn't verifiably found, the row says so — search "{name} {brokerage}" and confirm by photo/employer before connecting. Never guess a profile URL.

**What was deliberately excluded, so the next researcher doesn't re-tread:** franchise/platform-affiliated shops (eXp teams like Urban Nest, Side/The Agency-powered Clareo, Weichert-affiliated Muffley), 200+ agent independents (Atlanta Communities ~2,000, Ansley, Dorsey Alston, Beacham at luxury scale), solo-plus-staff shops below the 5-agent floor (Hirsh Real Estate, RealSource/Keen Team), and **all flat-fee/discount models** (Duffy, Clickit) — both a stated ICP disqualifier (tech-forward automation shops) and a conflict with flatsbo.
