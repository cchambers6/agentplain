# Founder credibility surface — how to use these files

**Date:** 2026-07-08. **Owner:** Conner (every artifact here goes out under his name; nothing in this folder is fleet-sent).
**Why this exists:** every prospect who opens a founder email looks Conner up before replying. The dept-marketing plan calls his profile "the least-managed surface in the funnel" (`docs/departments/2026-07-03/marketing/03-outbound-support-collateral.md`). These six files close that gap before the next Monday send block (Monday, 2026-07-13).

## What goes where

| File | Destination | Who acts |
|---|---|---|
| `01-founder-bio-variants.md` | 25-word → X/Twitter bio field. 75-word → LinkedIn About section. 200-word → site "About Conner" section (held for Conner per the SEO/AEO wave; this is the draft that unblocks it). | Conner pastes; site version lands via a follow-up eng PR once he approves the text |
| `02-linkedin-profile-audit.md` | Conner's own LinkedIn profile — a run-once checklist | Conner, ~30 minutes |
| `03-why-us-one-pager.md` | PDF export (doc pipeline, Heritage tokens). Linked from reply emails and call confirmations — never attached to a cold first touch (outreach-kit rule, `docs/outreach/2026-07-03-design-partner-kit/01-first-touch-emails.md`) | Marketing exports; Conner links |
| `04-founders-note-for-first-touch.md` | Optional warm paragraph for any first-touch email — slots where `{{custom-hook}}` sits in the outreach-kit variants | Conner, at send time, per prospect |
| `05-google-yourself-audit.md` | Run against live Google results — what a prospect finds and what to fix first | Conner, ~20 minutes |

## Order of operations (before Monday 2026-07-13)

1. **Wednesday–Thursday:** run `05` (Google-yourself audit). It decides how urgent the rest is and surfaces anything this folder didn't predict.
2. **Thursday:** work through `02` (LinkedIn checklist) top to bottom, pasting the About text from `01` as you go. The banner PNG comes from creative-router per the dept-marketing spec; everything else needs no one but you.
3. **Friday:** marketing exports `03` to PDF and drops the link into the `/operator/outreach` prospect rows. Confirm `hello@agentplain.com` actually reaches you before the PDF ships with it printed on the page.
4. **Monday:** send. Use `04` where a prospect deserves the warmer open. The one-pager goes out only in replies and confirmations.
5. **After the first replies:** the site About section from `01` lands via an eng PR. It is the least time-sensitive piece; a strong LinkedIn covers the who-is-this-guy moment on day one.

## Ground rules every artifact here follows

- Every claim is defensible today: no customer counts, no exits, no funding, no invented credentials. Zero named customers is stated plainly wherever the subject comes up.
- The model vendor is invisible (`feedback_model_vendor_invisible_on_customer_surfaces`). No provider names anywhere in this folder.
- Voice per `docs/brand/voice-guidelines-2026-06-19.md` and the do/don't in `docs/marketing/deep-dive-2026-07-02/05-brand-voice-do-and-dont.md`. The across-the-counter test settles arguments.
- lowercase **agentplain**, always. The unit is **the fleet**, never an agent count.
- Control line wherever the product is described: the fleet drafts; Conner's prospects approve and send.
