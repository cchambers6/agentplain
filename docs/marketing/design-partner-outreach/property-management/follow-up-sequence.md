# Property-Management — Follow-Up Sequence

Four touches after the first-contact email. Each varies the angle — never repeat a hook.
60–110 word bodies. Subject lines lowercase-leaning, plain, ≤ 8 words, no exclamation.
Sign as Conner Chambers, founder. Use only ground-truth claims.

Cadence: T+3d → T+7d → T+14d → T+21d (breakup).
