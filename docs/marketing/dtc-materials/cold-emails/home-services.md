# Cold email sequence — home services (roofing · HVAC · plumbing · electrical · GC)

**Vertical:** `home-services` · **Tier:** Partner ($299 → $199/seat) · **Trial:** 7-day, card at signup, first month free, 14-day money-back.
**Voice law:** `docs/marketing/CREATIVE_PACK_GROUND_TRUTH.md` + `docs/marketing/brand-voice-scenario-library.md`.
**Recipient:** owner / office manager of a trades shop. Soft ask first. Short, plain bodies, owner-to-owner. Subject lines lowercase-leaning, ≤8 words, no exclamation, no emoji.
**Honesty spine:** the fleet drafts; the owner approves and sends. Live integrations = email + Google Calendar + QuickBooks. Never imply auto-send. No price quoted except the published $199/seat.

**Cadence:** 5 touches over 14 days — T1 day 0 · T2 day 3 · T3 day 6 · T4 day 10 · T5 day 14.

**Personalization tokens:**
- `{{FirstName}}` — recipient first name. Note: if missing, drop the greeting line entirely; never "Hi there."
- `{{CompanyName}}` — the shop. Use once, naturally.
- `{{Trade}}` — roofing / HVAC / plumbing / electrical / GC. Note: tunes the scene (roofing → supplement; HVAC/plumbing/electrical → estimate + after-hours calls). If unknown, use generic "trades shop" and the estimate scene.
- `{{City}}` — metro. Note: only use if it adds a real signal (recent storm, season); never as filler.

## Sequence notes

- **Soft ask escalation:** T1–T2 ask only "worth a look?" T3 offers a walkthrough. T4 introduces the trial/price. T5 releases. Never push a demo before value is shown.
- **Every touch** names the human's control (you approve and send) at least once — it's the differentiator, not a disclaimer.
- **Never** quote an estimate amount, promise a booked slot, or imply the fleet texts/sends. The only numbers allowed: $199/seat, $50,000+/yr, ~$4,167/mo, 14×–21×, first month free.
- **No banned words.** Run T1–T5 against the five quick tests before send.
