# Pattern: ratified-frame preamble

**Group:** orchestration/governance · **Seeded by:** the "Ratified frame" block opening every 2026-07 orchestrator pass.

> Full paste-in preamble: **[`../governance/ratified-frame-preamble.md`](../governance/ratified-frame-preamble.md)**.

## When to use
Any prompt that asks an agent to plan, prioritize, cut, or decide — "if the pass produces decisions, it gets the preamble."

## The primitive
Prepend a standing block that states: the **model default** (back-to-opus), the **CEO's biggest lever** (first 5 GA-RE sends), the **kill list** (#1–#5, #3 overridden — flatsbo stays live), the **load-bearing positioning overrides** (audience = local businesses; service-layer-on-Claude; model/vendor invisible; customer vocab; no quick fixes), and **output discipline** (decisions > narrative, cite artifacts, ≥1 deliverable). The agent then reasons *inside* the guardrails instead of rediscovering them.

## Guardrails
- Refresh the dated facts, keep the shape. Don't add new standing rules here. One source of truth per fact — flag conflicts for Conner.
