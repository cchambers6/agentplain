# Counsel-handoff packets — index

**For counsel:** each file contains (1) live literal triggers sentinel fires on today, (2) candidate literal triggers awaiting phrase-by-phrase red-line, (3) candidate regex triggers, (4) counsel-reference regulatory text, and (5) questions for counsel with the most-ambiguous rules surfaced first.

Every corpus below is DRAFT. Sentinel does not flip live for a vertical until counsel signs off AND ops enables it via `COMPLIANCE_CORPUS_COUNSEL_REVIEWED`.
